  • Download Plugin

    Thank you for downloading our plugin. You saved your time!

    You can download the plugin here page. Please unpack the downloaded archive and zip archivate "azericard-payment-gateway-woocommerce" directory, that is plugin. Upload plugin archive into your website's /wp-content/plugins/ directory

    Alternatively you can add the plugin thru admin panel Plugins/Add New and browse there plugin archive.

  • WooCommerce Installation

    Azericard Payment Gateway is extended plugin for WooCommerce plugin. Hence, it is necessary to install WooCommerce plugin at first.

  • Activation

    After installation the plugin, it is necessary to activate it from: Plugins/Installed Plugins and click on "Activate" under Azericard plugin name

  • Settings

  • Plugin Settings

    Go to WooCommerce/Settings/Payments/Azericard for the plugin settings

  • Main Settings

    Please have a look at the screenshot below for the main settings of the Azericard Payment Gateway:

    • 1. Enable/Disable. To enable/disable Azericard Payment Gateway.
    • 2. Title. It is displayed on the checkout page
    • 3. Description. It is displayed on the checkout page.
    • 4. Sale mode. Payment mode option.
      • 1) Test mode - This is test mode, you should run in test mode before completly ensure that everything works fine
      • 2) Production mode - You card is charged in production mode. Please ask Azericard to switch your merchant account to production, before you switch on plugin settings
    • 5. Azericard url. A URL where payment query will be sent (Obtain from Azericard technical support).
    • 6. Terminal. Terminal number for your merchant account (Obtain from Azericard technical support).
    • 7. Key for sign. Key for sign (Obtain from Azericard technical support).
    • 8. Return page. Url of the page where your customers will be returned.
  • Azericard Processing Center

    You should get all necessary credentials and informations from Azericard technical support tsupport@azericard.com to enter into plugin settings

  • Payment

  • "Payments" Page

    After correct entry of all necessary settings you would see Azericard payment method on your website's checkout page.

  • Azericard Payment Method

    After checkout page, customer is sent to Azericard page where customer have to enter credit card details (in the test mode Azericard support provides special test credit card in testing purposes).

  • Orders Details

    If everything entered correctly, customer can print and return to website.

  • Orders in Admin Panel

    If transaction successfully passed, you will see new order with "Processing" status. Otherwise with status "Pending Payment", and later will be changed to "Cencelled"

  • Order Refund

  • Order Refund

    Azericard processing center accept total refunds of order in case you need, and this plugin support it.

    To refund administrator of website should go to WooCommerce/Orders/ click on appropriate order, in landed page click "Refund" button and fill in the refund form.

    After successful refund in "Order notes" will appear "Success! message - The payment was refunded.", and order status will be changed to "Refunded"

    Otherwise you will see this message "Something wrong. The payment wasn't refunded. Please, try later.". In that case you can contact Azericard technical support for the refund status.

    Note: At the moment Azericard processing center doesn't allow to refund more that total order sum.
